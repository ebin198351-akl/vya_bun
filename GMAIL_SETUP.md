# Gmail 应用密码设置指南

## 问题说明

Gmail 出于安全考虑，不允许使用普通密码通过 SMTP 发送邮件。需要使用 **应用密码（App Password）**。

## 设置步骤

### 1. 开启两步验证（如果还没开启）

1. 访问 [Google 账户设置](https://myaccount.google.com/)
2. 进入 **安全性** 页面
3. 找到 **两步验证**，点击进入
4. 按照提示开启两步验证（需要手机号）

### 2. 创建应用密码

1. 在 **安全性** 页面，找到 **应用密码**（在两步验证下方）
2. 如果看不到"应用密码"选项：
   - 确保已开启两步验证
   - 可能需要等待几分钟后刷新页面
3. 点击 **应用密码**
4. 选择应用：**邮件**
5. 选择设备：**其他（自定义名称）**
6. 输入名称：`Vya's Kitchen`
7. 点击 **生成**
8. **重要**：复制生成的 16 位密码（格式类似：`abcd efgh ijkl mnop`，去掉空格后是 16 位）

### 3. 更新代码

将 `send_email.py` 第 32 行的密码替换为应用密码：

```python
sender_password = os.getenv("GMAIL_APP_PASSWORD", "你的16位应用密码（去掉空格）")
```

例如，如果应用密码是 `abcd efgh ijkl mnop`，则使用：
```python
sender_password = os.getenv("GMAIL_APP_PASSWORD", "abcdefghijklmnop")
```

### 4. 重启服务器

```bash
python server.py
```

### 5. 测试

运行测试脚本：
```bash
python test_email.py
```

或者在网页上提交联系表单测试。

## 注意事项

- 应用密码是 16 位字符（可能显示为 4 组，每组 4 位，需要去掉空格）
- 不要分享应用密码给他人
- 如果应用密码泄露，可以在 Google 账户设置中删除并重新生成

## 常见问题

**Q: 找不到"应用密码"选项？**
A: 确保已开启两步验证，可能需要等待几分钟后刷新页面。

**Q: 仍然提示认证失败？**
A: 
- 检查应用密码是否正确（16 位，无空格）
- 确认邮箱地址正确：`vya2025.kitchen@gmail.com`
- 尝试重新生成应用密码

**Q: 可以使用其他邮箱吗？**
A: 可以，但 Gmail 是最常用的。其他邮箱（如 Outlook）也需要类似的应用密码设置。
